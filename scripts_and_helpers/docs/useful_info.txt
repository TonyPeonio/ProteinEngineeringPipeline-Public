pLDDT (predicted Local Distance Difference Test): Per-residue confidence score (0-100)
PAE (Predicted Aligned Error): Pairwise residue position confidence (in Ångstroms)
PTM (predicted TM-score): Overall fold quality assessment
iPTM (predicted interface TM-score): Interface quality in multimers
Ranking confidence: Single score used to rank multiple models
